# 📸 ПОРАДИ ДЛЯ КРАЩОГО РОЗПІЗНАВАННЯ ДОКУМЕНТІВ

**Якісне фото = Точний переклад = Правильна відповідь**

---

## ✅ ЩО РОБИТИ

### 1. 💡 Освітлення

```
✅ Використовуйте денне світло біля вікна
✅ Увімкніть верхнє освітлення в кімнаті
✅ Тримайте джерело світла збоку або ззаду
✅ Використовуйте спалах тільки якщо темно
```

### 2. 📐 Положення телефону

```
✅ Тримайте телефон паралельно документу
✅ Камера має дивитися прямо зверху
✅ Всі 4 кути документа мають бути видні
✅ Використовуйте обидві руки для стабільності
```

### 3. 🔍 Фокус

```
✅ Натисніть на екран для фокусування
✅ Зачекайте 1-2 секунди поки камера сфокусується
✅ Переконайтесь що текст чіткий перед зйомкою
✅ Зробіть тестове фото і перевірте
```

### 4. 📄 Підготовка документу

```
✅ Розправте документ на рівній поверхні
✅ Приберіть сторонні предмети з кадру
✅ Використовуйте темний фон для контрасту
✅ Видаліть скріпки та скоби
```

---

## ❌ ЧОГО УНИКАТИ

### 1. 🌑 Погане освітлення

```
❌ Не робіть фото в темній кімнаті
❌ Уникайте тіней на документі
❌ Не фотографуйте проти світла
❌ Уникайте відблисків від ламп
```

### 2. 📐 Неправильний кут

```
❌ Не фотографуйте під кутом
❌ Не тримайте телефон занадто близько
❌ Не фотографуйте "з руки" без опори
❌ Уникайте викривлення перспективи
```

### 3. 🌫️ Розмиття

```
❌ Не рухайте телефон під час зйомки
❌ Не фотографуйте на ходу
❌ Уникайте тремтіння рук
❌ Не робіть фото в транспорті
```

### 4. 🗑️ Зайві об'єкти

```
❌ Не фотографуйте з руками в кадрі
❌ Приберіть ручки, ключі, телефони
❌ Не фотографуйте на ліжку/дивані
❌ Уникайте тіней від предметів
```

## 📱 КОРИСНІ ДОДАТКИ

### Для сканування:

1. **Adobe Scan** (безкоштовно)
   - Автоматичне вирівнювання
   - Покращення якості
   - Експорт в PDF

2. **CamScanner** (безкоштовно)
   - Розумне кадрування
   - Фільтри для документів
   - Хмарне зберігання

3. **Microsoft Lens** (безкоштовно)
   - Інтеграція з Office
   - Розпізнавання тексту
   - Безкоштовно

## 🔍 ПЕРЕВІРКА ЯКОСТІ

Перед відправкою перевірте:

```
✅ Текст чіткий і читається
✅ Всі символи видні (ä, ö, ü, ß)
✅ Немає відблисків на тексті
✅ Документ займає 80-90% кадру
✅ Немає сторонніх предметів
✅ Кути документа не обрізані
```

## 🎯 ПРИКЛАДИ

### ✅ ПРАВИЛЬНО:

```
Фото 1:
```

### ❌ НЕПРАВИЛЬНО:

```

- Телефон тремтів в руці
- Документ зім'ятий, кути загорнуті
- Відблиски від спалаху на тексті
- Частина тексту обрізана
```

## 💡 ЛАЙФХАКИ

### 1. 📚 Використовуйте книгу як підставку
```
Покладіть документ на книгу → 
Телефон притисніть до книги зверху → 
Зробіть фото без тремтіння
```

### 2. 🪟 Вікно = найкраще світло
```
Покладіть документ біля вікна → 
Світло падає збоку → 
Немає тіней та відблисків
```

### 3. 📱 Режим "Документ" в телефоні
```
Багато телефонів мають режим "Document" → 
Автоматично покращує контраст → 
Вирівнює перспективу
```

### 4. 🔦 Спалах тільки в темряві
```
Вдень → вимкніть спалах
Вночі → увімкніть, але тримайте телефон далі
Краще → увімкніть світло в кімнаті
```

## 🚀 ШВИДКА ПЕРЕВІРКА

Перед відправкою в бот:

```
1. Відкрийте фото в галереї
2. Наблизьте текст (zoom)
3. Чи читаються символи ä, ö, ü, ß?
4. Чи немає розмитих ділянок?
5. Чи всі рядки видні?

Якщо ТАК на всі питання → Відправляйте!
Якщо НІ → Зробіть нове фото
```

## 📊 СТАТИСТИКА

### Вплив якості фото на точність:

| Якість фото | Точність OCR | Точність перекладу |
|-------------|--------------|-------------------|
| **Відмінна** | 95-99% | 95-98% |
| **Добра** | 85-95% | 90-95% |
| **Задовільна** | 60-85% | 70-85% |
| **Погана** | 30-60% | 40-60% |

**Висновок:** Якісне фото покращує точність на 50-70%!
