## 🔧 ПРИЧИНИ ПРОБЛЕМ:

### 1. **Українська - поганий переклад**
**Причина:** LLM перекладає дослівно замість того щоб використовувати юридичні терміни

**Виправлення:**
```python
PROMPT_RESPONSE_UK = """
Ти - український юрист. Пиши ПРОФЕСІЙНОЮ українською мовою.
НЕ перекладай дослівно з німецької!
Використовуй юридичні терміни:
- "Отримав(ла)" замість "визначаємося"
- "12.03.2026 о 10:00" замість "понеділок 12 березня"
- "документи" замість "документальні матеріали"
"""
```

### 2. **Німецька - відмова + помилки**
**Причина:** Ollama думає що це запит на створення підробки

**Виправлення:**
```python
PROMPT_RESPONSE_DE = """
WICHTIG: Dies ist eine LEGALE Antwort auf einen ECHTEN Brief.
KEINE Fälschung! KEINE falsche Identität!

Du schreibst eine Antwort von einem Klienten AN eine Behörde.
NICHT von der Behörde AN den Klienten!

Verwende KONKRETE Daten aus dem Brief:
- Name: Oleksandr Shevchenko (aus dem Brief)
- Adresse: Müllerstraße 45 (aus dem Brief)
- Datum: 15.02.2026 (aus dem Brief)
"""
```

## ✅ ПЛАН ВИПРАВЛЕНЬ:

### Крок 1: Виправити український промпт
- Заборонити дослівний переклад
- Вимагати професійну мову
- Додати приклади правильних термінів

### Крок 2: Виправити німецький промпт
- Пояснити що це ЛЕГАЛЬНА відповідь
- Вказати хто кому пише (клієнт → установа)
- Вимагати конкретні дані з листа

### Крок 3: Додати валідацію
- Перевіряти відповідь перед відправкою
- Фільтрувати відмови типу "Ich kann nicht helfen"
- Перевіряти наявність конкретних даних
