# 📄 Багатосторінкові Документи - Інструкція

## 🎯 Як Це Працює

### Для Користувача:

1. **Натискає** "📤 Завантажити лист"
2. **Бот показує** інструкцію для багатосторінкових документів
3. **Користувач надсилає** перше фото
4. **Бот розпізнає** та запитує "Чи є ще сторінки?"
5. **Користувач обирає:**
   - "📄 Надіслати ще сторінку" → надсилає наступне фото
   - "✅ Все, аналізуй" → бот починає аналіз

### Для Бота:

```python
# Стани діалогу
WAITING_FOR_LETTER = 5       # Очікування фото
WAITING_FOR_MORE_PAGES = 7   # Очікування рішення (ще сторінки / аналіз)

# Зберігання даних
context.user_data['letter_photos'] = []  # Список фото
context.user_data['letter_text'] = ''     # Об'єднаний текст
```

```
Користувач: 📤 Завантажити лист

4️⃣ Коли всі фото надіслані - натисніть "✅ Все, аналізуй"

Користувач: 📄 Надіслати ще сторінку
```

## 🔧 Технічні Деталі

### Зберігання:

```python
# Кожне фото зберігається
context.user_data['letter_photos'] = [
    'uploads/chat_id_20260228_001.jpg',
    'uploads/chat_id_20260228_002.jpg',
    ...
]

# Текст об'єднується з маркерами сторінок
context.user_data['letter_text'] = '''
--- СТОРІНКА 1 ---
[текст першої сторінки]

--- СТОРІНКА 2 ---
[текст другої сторінки]
'''
```

### Обробка:

1. **OCR кожної сторінки** окремо
2. **Об'єднання тексту** з маркерами
3. **Аналіз всього тексту** як одного документу
4. **Відповідь** з урахуванням всіх сторінок

## 📊 Обмеження

- **Максимум сторінок:** 10 (можна збільшити)
- **Максимум тексту:** 4000 символів (обмеження Telegram)
- **Час очікування:** 5 хвилин між фото
