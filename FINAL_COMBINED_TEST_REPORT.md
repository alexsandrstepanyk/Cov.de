## ❌ ГОЛОВНІ ПРОБЛЕМИ

### Проблема #1: Виявлення Параграфів (40% на великих)

**Симптоми:**
- Test #2 (Inkasso): 1/5 параграфів знайдено
- Test #4 (Gericht): 1/5 параграфів знайдено

**Причина:**
Regex не підтримує формат "BGB § 286" (тільки "§ 286 BGB")

**Фікс:**
```python
# Поточний (не працює):
r'§\s*\d+\s*(?:BGB|SGB|ZPO|AO)'

# Новий (працює):
r'(?:§\s*\d+\s*(?:BGB|SGB|ZPO|AO)|(?:BGB|SGB|ZPO|AO)\s*§\s*\d+)'
```

---

### Проблема #2: Класифікація Великих Документів

**Симптоми:**
- Test #2 (3,656 chars): False (очікувалось True)
- Test #4 (3,739 chars): False (очікувалось True)

**Причина:**
- Можливе переповнення маркерами
- Відсутність "length bonus" для довгих текстів

**Фікс:**
```python
# Додати bonus для довгих документів
length_bonus = 1 if len(text) > 3000 else 0
official_score += length_bonus
```

---

### Проблема #3: Fraud Detection не блокує

**Симптоми:**
- Test #5 (Fake Finanzamt): fraud_score=16, risk=high
- АЛЕ: is_document=True

**Причина:**
Fraud detection не інтегровано в check_if_document()

**Фікс:**
Вже створено в `quick_fixes.py`
