# 🚀 ШВИДКИЙ СТАРТ - Windows 10

## Один клік - і все працює!

---

## 📥 КРОК 1: Завантаж файл

1. Скопіюй файл **`INSTALL_WINDOWS.bat`** на робочий комп'ютер
2. Збережи в будь-яке місце (наприклад, на Робочий стіл)

---

## ▶️ КРОК 2: Запусти

1. **Права клік** на `INSTALL_WINDOWS.bat`
2. Обери **"Run as administrator"** (Запуск від імені адміністратора)
3. Чекай 5-10 хвилин

**Все!** 🎉 Скрипт сам:
- ✅ Встановить Python 3.9
- ✅ Встановить Git
- ✅ Завантажить проект
- ✅ Встановить всі залежності
- ✅ Встановить Tesseract OCR
- ✅ Налаштує автозапуск
- ✅ Створить ярлики

## ▶️ ПЕРШИЙ ЗАПУСК:

Після завершення установки:

**Варіант 1: Автоматично**
- Бот запуститься сам при наступному вході в Windows

**Варіант 2: Вручну**
1. Відкрий `C:\gov-de-bot`
2. Двічі клікни на **`start-bot.bat`**
3. Готово!

## 🛑 ЯК ЗУПИНИТИ:

**Варіант 1: Task Manager**
1. Ctrl+Shift+Esc
2. Details
3. Знайди `python.exe`
4. End Task

**Варіант 2: Вимкнути автозапуск**
1. Task Scheduler
2. Знайди "Gov.de Bot"
3. Disable

---

## 🔧 ЯКЩО ЩОСЬ НЕ ТАК:

### Бот не запускається:
1. Відкрий `C:\gov-de-bot\logs\client_bot.log`
2. Перевір помилки
3. Напиши мені - допоможу!

### Python не встановився:
1. Завантаж вручну: https://www.python.org/downloads/
2. Встанови Python 3.9.13
3. Запусти скрипт знову

### Telegram не відповідає:
1. Перевір інтернет
2. Перевір токен в `.env`
3. Перезапусти бота

---

## 📞 ПІДТРИМКА:

Якщо щось не працює:
1. Зроби скріншот помилки
2. Відкрий `C:\gov-de-bot\logs\client_bot.log`
3. Надішли мені
