## ⚠️ ПРОБЛЕМИ

### 1. Contact Person Extraction (10%)

**Проблема:**
```
Вхідне: "Im Auftrag\n\nMaria Schmidt\nBeraterin"
Витягнуто: {'firstname': 'Im', 'lastname': 'Auftrag'}
Очікувалось: {'firstname': 'Maria', 'lastname': 'Schmidt'}
```

**Причина:**
- "Im Auftrag" сприймається як ім'я
- Складні підписи з титулами

**Рішення:**
- Додати фільтр для "Im Auftrag", "i.A.", "i.V."
- Покращити regex для підписів

### 2. response_generation (66%)

**Проблема:**
```
Деякі листи не мають достатньо контексту
→ generate_response_smart_improved повертає <100 символів
```

**Причина:**
- Короткі листи (<500 символів)
- Відсутність ключових слів для класифікації

**Рішення:**
- Додати fallback шаблони
- Покращити класифікацію
