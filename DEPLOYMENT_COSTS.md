## 🚀 ШВИДКИЙ ДЕПЛОЙ НА VPS

### 1. Купити VPS (5 хв)

```bash
# Hetzner Cloud
https://console.hetzner.cloud/

# Обрати CPX11 (€5.50/міс)
# Обрати location (Nuremberg/Helsinki)
# Створити сервер
```

### 2. Підключитися (1 хв)

```bash
ssh root@your-server-ip
```

### 3. Встановити Docker (5 хв)

```bash
curl -fsSL https://get.docker.com | sh
```

### 4. Клонувати проект (2 хв)

```bash
git clone <your-repo> gov-de
cd gov-de
```

### 5. Налаштувати .env (2 хв)

```bash
cp .env.example .env
nano .env
# Відредагувати токени
```

### 6. Запустити (5 хв)

```bash
docker compose up -d
```

### 7. Перевірити (1 хв)

```bash
docker compose ps
docker compose logs -f
```

**Всього: 20-25 хвилин**
