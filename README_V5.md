## 🎯 ОГЛЯД

Gov.de Bot v5.0 - це професійна система аналізу німецьких юридичних листів з **повною базою всіх німецьких законів** (307,496 параграфів) та **Ollama LLM** як "мозком" системи.

### Ключові можливості:

- ✅ **307,496 параграфів** - ВСІ німецькі кодекси та закони
- ✅ **Ollama Llama 3.2 3B** - розумний аналіз листів
- ✅ **RAG пошук** - миттєвий пошук по всій базі законів
- ✅ **Точні цитати** - посилання на конкретні параграфи
- ✅ **Двомовність** - українська + німецька відповіді
- ✅ **DIN 5008** - професійні німецькі листи
- ✅ **Перехресна перевірка** - декілька джерел для точності

## 🚀 ВСТАНОВЛЕННЯ

### Вимоги:
- Python 3.9+
- 2GB+ вільного місця
- Ollama (встановлюється автоматично)

### Швидке встановлення:

```bash
# 1. Клонувати репозиторій
cd /Users/alex/Desktop/project/Gov.de

# 2. Встановити залежності
pip3 install -r requirements.txt

# 3. Встановити Ollama
brew install ollama  # macOS
ollama pull llama3.2:3b

# 4. Завантажити базу законів (вже в репозиторії)
# data/german_laws_complete/ - 903MB

# 5. Запустити бота
python3 src/bots/client_bot.py
```

## 📖 ВИКОРИСТАННЯ

### Реєстрація в Telegram:
1. Знайти бота: `@ClientCovde_bot`
2. Натиснути `/start`
3. Пройти реєстрацію

### Аналіз листа:
1. Надіслати фото німецького листа
2. Бот розпізнає текст (OCR)
3. LLM проаналізує (знайде параграфи)
4. RAG знайде相关法律 в базі
5. Бот надішле відповідь:
   - **Українська:** аналіз + рекомендації (6000+ символів)
   - **Німецька:** готовий лист (DIN 5008, 2000+ символів)
