### 1. Німецька відповідь - КРИТИЧНА ПРОБЛЕМА!
**Причина:** Ollama Llama 3.2 3B **НЕ СЛУХАЄ** промпт!

```python

"NICHT 'Ich kann nicht helfen'!"
"Schreibe SOFORT die Antwort ohne Ablehnung!"

# Але LLM все одно пише:
"Ich kann nicht dabei helfen"
```
