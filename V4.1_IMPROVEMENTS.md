    # Перевірка якості

        keyboard = [['❌ Перезробити', '✅ Відправити']]

## 📊 ІНТЕГРАЦІЯ В БОТА

### client_bot.py - Зміни:

1. **Імпорти:**
```python
from legal_dictionary import LEGAL_TRANSLATION_FIXES_EXTENDED
from statistics import log_event, create_reminder, check_organization, ...
```

2. **Об'єднання словників:**
```python
LEGAL_TRANSLATION_FIXES.update(LEGAL_TRANSLATION_FIXES_EXTENDED)
```

3. **Логування подій:**
```python
# Після обробки листа
log_event(
    chat_id=chat_id,
    event_type='letter_processed',
    letter_type=letter_type,
    quality_score=quality,
    ocr_confidence=confidence
)
```

4. **Створення нагадувань:**
```python
# Якщо є термін в листі
if deadline:
    create_reminder(
        chat_id=chat_id,
        letter_id=letter_id,
        reminder_text=f"Термін: {deadline_text}",
        reminder_date=deadline,
        reminder_time="09:00"
    )
```

5. **Перевірка організацій:**
```python
# Після аналізу
org_info = check_organization(law_info.get('organization', ''))
if org_info:
    if org_info['is_official']:
        result += f"\n\n✅ **Організація офіційна**"
    else:
        result += f"\n\n⚠️ **Організація підозріла**"
```
