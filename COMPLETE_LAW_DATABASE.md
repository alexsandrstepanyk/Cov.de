## 🛠️ Використання

### Python API:

```python
from build_law_database import (
    get_all_codes,
    get_paragraph,
    search_paragraphs,
    get_paragraphs_by_code,
    get_statistics
)

# Отримати всі кодекси
codes = get_all_codes()

# Отримати конкретний параграф
para = get_paragraph('BGB', '286')

# Пошук параграфів

# Отримати всі параграфи кодексу
bgbs = get_paragraphs_by_code('BGB')

# Статистика
stats = get_statistics()
```
