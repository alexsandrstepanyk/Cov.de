## 📌 Відомі Проблеми

### Обмеження WhatsApp API

- ⚠️ **Sandbox:** 1000 повідомлень/24год (безкоштовно)
- ⚠️ **Production:** $0.005/вхідне, $0.008/вихідне
- ⚠️ **Медіа:** До 16MB на файл
- ⚠️ **Кнопки:** Тільки текстове меню (немає inline кнопок)

### Технічні Обмеження

- ⚠️ **Ngrok:** Змінює URL при кожному запуску (free тариф)
- ⚠️ **Tesseract:** Потребує якісних фото для кращого OCR
- ⚠️ **Переклад:** Може потребувати перевірки для складних випадків
