## 🚀 Версія 4.0 (Березень 2026) - Anti-Fraud

### 🔍 Аналіз на шахрайство

#### 1. Перевірка контактних даних
- [ ] **Перевірка телефонів**
  - [ ] Порівняння з офіційними базами
  - [ ] Перевірка на платні номери (0900, 0180)
  - [ ] Перевірка країни (префікси +49, +44, +1)
  - [ ] API інтеграція з телефонними довідниками

- [ ] **Перевірка email**
  - [ ] Перевірка домену на офіційність
  - [ ] Блокlist підозрілих доменів (gmail, yahoo тощо)
  - [ ] Перевірка SPF/DKIM записів
  - [ ] API інтеграція з email валідаторами

- [ ] **Перевірка вебсайтів**
  - [ ] Перевірка домену на вік
  - [ ] Перевірка SSL сертифікату
  - [ ] Перевірка на офіційність (.de, .bund.de)
  - [ ] WHOIS lookup
  - [ ] API інтеграція з ScamAdviser

#### 2. Перевірка організацій
- [ ] **База офіційних організацій**
  - [ ] Jobcenter (з офіційними телефонами/сайтами)
  - [ ] Finanzamt (податкова)
  - [ ] Gericht (суди)
  - [ ] Stadt/Gemeinde (міста)
  - [ ] Inkasso (колекторські агенції - ліцензовані)

- [ ] **Реєстраційні номери**
  - [ ] Перевірка Handelsnummer (HRB)
  - [ ] Перевірка Steuernummer
  - [ ] Перевірка Vereinsregister
  - [ ] API інтеграція з Unternehmensregister

#### 3. Аналіз тексту на шахрайство
- [ ] **Ознаки терміновості**
  - [ ] "sofort", "umgehend", "24 Stunden"
  - [ ] "letzte Mahnung", "dringend"

- [ ] **Загрозливий тон**
  - [ ] "Gerichtsvollzieher", "Haftbefehl"
  - [ ] "Polizei", "Strafanzeige", "Verhaftung"
  - [ ] "Abschiebung" (для мігрантів)

- [ ] **Підозрілі методи оплати**
  - [ ] Western Union, MoneyGram
  - [ ] Bitcoin, криптовалюта
  - [ ] Geschenkkarte, Paysafecard

- [ ] **Граматичні помилки**
  - [ ] Типові помилки шахраїв
  - [ ] Неправильне звертання (Sie vs sie)

#### 4. Інтеграції з API
- [ ] **Офіційні бази даних**
  - [ ] Unternehmensregister (компанії)
  - [ ] Bundesanzeiger (офіційні публікації)
  - [ ] Insolvenzregister (банкрутства)
  - [ ] Gewerbezentralregister (бізнеси)

- [ ] **Anti-fraud сервіси**
  - [ ] ScamAdviser API
  - [ ] VirusTotal (для посилань)
  - [ ] Phone Validator API
  - [ ] Email Validator API

#### 5. Система оцінки ризику
- [ ] **Рівні ризику**
  - [ ] 🟢 Low (0-4 бали) - легітимно
  - [ ] 🟡 Medium (5-9 балів) - підозріло
  - [ ] 🔴 High (10+ балів) - ймовірне шахрайство

- [ ] **Фактори ризику**
  - [ ] Неправильний телефон (+2 бали)
  - [ ] Підозрілий email (+2 бали)
  - [ ] Загрозливий тон (+3 бали)
  - [ ] Терміновість платежу (+2 бали)
  - [ ] Криптовалюта (+5 балів)
  - [ ] Граматичні помилки (+2 бали)

#### 6. UI/UX покращення
- [ ] **Візуальні індикатори**
  - [ ] Кольорові індикатори ризику
  - [ ] Іконки перевірок
  - [ ] Progress bar аналізу

- [ ] **Детальні звіти**
  - [ ] PDF звіт з аналізом
  - [ ] Експорт результатів
  - [ ] Історія перевірок
