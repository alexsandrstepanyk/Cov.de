## ⚠️ ПРОБЛЕМИ

### 1. Український текст детектиться як валідний

**Причина:** Український текст має багато символів і не детектиться як сміття.

**Рішення:** Додати перевірку на наявність кирилиці:
```python
# Додати в TextValidator
CYRILLIC_CHARS = set('абвгдеєжзийклмнопрстуфхцчшщьюяїіё')

# В validate_text:
has_cyrillic = bool(CYRILLIC_CHARS & set(text_lower))
if has_cyrillic and not has_german_chars:
    issues.append('Знайдено кирилицю в німецькому тексті')
    score -= 20
```

### 2. Короткий текст не детектиться

**Причина:** Текст "Привіт світ" має 60% score = fair.

**Рішення:** Знизити поріг для коротких текстів або додати перевірку довжини:
```python
# Додати в validate_text
if len(text.split()) < 5:
    issues.append('Занадто мало слів')
    score -= 20
```

## 📈 РЕКОМЕНДАЦІЇ

### Пріоритет 1: Додати перевірку кирилиці

```python
# В src/advanced_ocr.py, TextValidator
CYRILLIC_CHARS = set('абвгдеєжзийклмнопрстуфхцчшщьюяїіё')

def validate_text(cls, text: str) -> Dict:
    ...
    has_cyrillic = bool(CYRILLIC_CHARS & set(text_lower))
    if has_cyrillic and not has_german_chars:
        issues.append('⚠️ Знайдено кирилицю в німецькому тексті')
        score -= 25
        return {
            'valid': False,
            'quality_score': max(0, score),
            'quality': 'poor',
            'issues': issues,
            'recommendations': [
                '📸 Зробіть фото кращої якості',
                '🔤 Переконайтесь що документ німецькою'
            ]
        }
```

### Пріоритет 2: Покращити детекцію коротких текстів

```python
# Додати перевірку кількості слів
if len(text.split()) < 5:
    issues.append('Занадто мало слів для аналізу')
    score -= 20
```

### Пріоритет 3: Додати перевірку на "кашу" з символів

```python
# Приклад: "baw homep kniehta (bypb nacka jobxam...)"
words = text.split()
nonsense_ratio = sum(1 for w in words if len(w) > 5 and not any(c in w for c in 'aeiouäöü')) / len(words)
if nonsense_ratio > 0.5:
    issues.append('Багато незрозумілих слів')
    score -= 25
```
