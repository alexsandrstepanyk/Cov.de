# 🧪 АВТОМАТИЧНЕ ТЕСТУВАННЯ TELEGRAM БОТА

## 📋 ІНСТРУКЦІЯ

### Крок 1: Отримати Bot Token та Chat ID

**Bot Token:**
1. Відкрити Telegram
2. Знайти @BotFather
3. Надіслати `/mybots`
4. Обрати свого бота (@ClientCovde_bot)
5. Скопіювати токен (наприклад: `8594681397:AAE7Y2OsQI8DOAK44Subfw8NwAf0ITgtqY0`)

**Chat ID:**
1. Відкрити Telegram
2. Знайти свого бота
3. Надіслати будь-яке повідомлення
4. Відкрити: https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates
5. Знайти `"chat":{"id":123456789,...}` - це ваш Chat ID

## 📊 ЩО ТЕСТУЄТЬСЯ:

### 1. **Якість відповідей:**
- ✅ Наявність української мови

### 2. **Продуктивність:**
- ⏱️ Час обробки одного листа
- ⏱️ Загальний час тестування

### 3. **Повнота:**
