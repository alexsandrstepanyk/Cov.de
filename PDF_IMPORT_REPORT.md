## 🔧 ТЕХНІЧНІ ДЕТАЛІ

### Скрипт імпорту:
```bash
python3 import_pdf_laws_to_rag.py
```

### Що робить скрипт:
1. Витягує текст з PDF (pdfminer)
2. Парсить параграфи (§ X)
3. Додає до ChromaDB з метаданими:
   - `law`: Код кодексу (BGB, AO, тощо)
   - `paragraph`: Номер параграфу (§ 286)
   - `source`: "PDF"
   - `added_at`: Дата додавання

- **Колекція:** `german_laws`
