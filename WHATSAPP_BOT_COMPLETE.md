## 🚀 Швидкий старт

### 1. Встановлення залежностей

```bash
cd /Users/alex/Desktop/project/Gov.de
pip3 install twilio flask python-dotenv
```

### 2. Налаштування Twilio

1. Зареєструйтесь на [Twilio](https://www.twilio.com/try-twilio)
2. Отримайте **Account SID** та **Auth Token**
3. Активуйте WhatsApp Sandbox

### 3. Конфігурація

Створіть `.env` файл:

```bash
cp src/whatsapp/.env.example .env
nano .env
```

Заповніть своїми даними:

```env
TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxx
TWILIO_AUTH_TOKEN=your_auth_token
TWILIO_WHATSAPP_NUMBER=whatsapp:+14155238886
```

### 4. Запуск

```bash
# Спосіб 1: Скрипт
bash run_whatsapp_bot.sh

# Спосіб 2: Безпосередньо
python3 src/whatsapp/whatsapp_bot.py

# Спосіб 3: Background
nohup python3 src/whatsapp/whatsapp_bot.py > logs/whatsapp_bot.log 2>&1 &
```

### 5. Налаштування вебхука

**Для локальної розробки (ngrok):**

```bash
brew install ngrok
ngrok http 5000
```

Скопіюйте URL з ngrok в Twilio Console:
```
https://xxx.ngrok.io/whatsapp
```

## 📱 Як користуватися

### Сценарій 1: Аналіз юридичного листа

```
1. Надішліть "1" в WhatsApp
2. Бот попросить фото
3. Надішліть фото Jobcenter листа
4. Отримайте:
   ✅ Розпізнаний текст
   ✅ Переклад українською
   ✅ Знайдені параграфи (§ 59 SGB II, § 309 SGB III)
   ✅ Готову відповідь
```

### Сценарій 2: Багатосторінковий документ

```
1. Надішліть фото першої сторінки
2. Бот: "Сторінка 1 оброблена. Чи є ще сторінки?"
3. Надішліть наступні фото
4. Напишіть "✅ Все, аналізуй"
5. Отримайте об'єднаний аналіз всіх сторінок
```

### Сценарій 3: Виявлення шахрайства

```
1. Надішліть фото підозрілого листа
2. Бот розпізнає маркери шахрайства
3. ⚠️ УВАГА: Можливе шахрайство!
4. Поради щодо безпеки
```

## 🔧 Вирішення проблем

### Бот не запускається

```bash
# Перевірка логів
tail -f logs/whatsapp_bot.log

# Перевірка портів
lsof -i :5000

# Перезапуск
pkill -f whatsapp_bot.py
python3 src/whatsapp/whatsapp_bot.py &
```

### Вебхук не працює

```bash
# Перевірка ngrok
curl https://your-ngrok-url.ngrok.io/health

# Перевірка вебхука
curl "https://your-ngrok-url.ngrok.io/webhook?hub.mode=subscribe&hub.verify_token=gov_de_2026&hub.challenge=test"
```

### Помилки Twilio

1. Перевірте баланс Twilio Console
2. Переконайтесь що Sandbox активований
3. Перевірте логи в Twilio Console → Messaging → Logs

## 💰 Вартість Twilio WhatsApp API

### Sandbox (тестування)

- **Безкоштовно** $15 кредит при реєстрації
- **Обмеження:** 1000 повідомлень/24год
- **Номер:** Спільний sandbox номер

### Production (продакшен)

- **Вхідні повідомлення:** $0.005/шт
- **Вихідні повідомлення:** $0.008/шт
- **Місячна плата:** $0 (тільки за використання)

**Приклад розрахунку:**
```
1000 користувачів × 5 повідомлень/міс = 5000 повідомлень
5000 × $0.005 = $25/міс
```
