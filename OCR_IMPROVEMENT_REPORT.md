### 1. 🔍 TEXT VALIDATOR

**Створено новий клас `TextValidator`:**

```python
class TextValidator:
    """Валідація якості OCR тексту."""
    
    # Перевірка на німецькі символи
    GERMAN_CHARS = set('äöüÄÖÜß')
    
    # Патерни "сміття" (garbage text)
    GARBAGE_PATTERNS = [...]
    
    # Очікувані слова для Jobcenter
    EXPECTED_WORDS = {...}
```

**Функції валідації:**

```python
validate_text(text) → Dict:
  ✅ Перевірка на німецькі символи
  ✅ Виявлення "сміття" в тексті
  ✅ Перевірка очікуваних слів
  ✅ Аналіз довжини слів
  ✅ Оцінка якості (good/fair/poor)
```

**Приклад використання:**

```python
# Хороший текст
text = "Ihre Kundennummer: BG123456\nEinladung zum Gespräch"
validation = TextValidator.validate_text(text)
# Результат: {'valid': True, 'quality_score': 85, 'quality': 'good'}

# Поганий текст (приклад з бота)
text = "baw homep kniehta (bypb nacka jobxam...)"
validation = TextValidator.validate_text(text)
# Результат: {'valid': False, 'quality_score': 40, 'quality': 'poor'}
```

### 3. 🔍 ІНТЕГРАЦІЯ ВАЛІДАЦІЇ В OCR

**Додано валідацію до головного методу:**

```python
def recognize(self, image_path: str, lang: str = 'de') -> Dict:
    result = {
        'text': '',
        'engine': 'none',
        'confidence': 0,
        'quality': {},
        'validation': {},        # ← НОВЕ
        'recommendations': []
    }
    
    # ... розпізнавання ...
    
    # 🔍 ВАЛІДАЦІЯ ТЕКСТУ
    if best_text:
        result['validation'] = TextValidator.validate_text(best_text)
        
        # Додаємо поради
        if result['validation'].get('recommendations'):
            result['recommendations'].extend(
                result['validation']['recommendations']
            )
        
        # Логування попереджень
        if not result['validation']['valid']:
            logger.warning(f"⚠️ Текст не пройшов валідацію")
```

