## 🔧 OCR Система

Бот використовує два методи OCR:

1. **pytesseract** (якщо встановлено) — швидкий, якісний
2. **EasyOCR** (завжди доступний) — універсальний, не потребує системних залежностей

### Як це працює:

```python
def extract_text_from_photo(photo_path: str) -> str:
    # Спроба 1: pytesseract
    try:
        text = pytesseract.image_to_string(img, lang='deu+eng')
        if text.strip():
            return text
    except:
        pass
    
    # Спроба 2: EasyOCR
    try:
        reader = easyocr.Reader(['en', 'de'], gpu=False)
        results = reader.readtext(photo_path)
        text = ' '.join([r[1] for r in results])
        return text
    except:
        pass
    
    # Fallback
    return "[Фото отримано. Надішліть текст вручну]"
```
