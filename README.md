# 🇩🇪 Gov.de v4.0 - Розумний Аналізатор Юридичних Листів

**Версія 4.0** - Лютий 2026  
**Статус:** ✅ **ГОТОВО ДО ЗАПУСКУ** (98%)

Професійна система Telegram-ботів для аналізу німецьких юридичних листів з **розумним перекладом**, **багатосторінковою підтримкою** та **автоматичною генерацією відповідей**.

---

## 🎯 МОЖЛИВОСТІ

### 📸 Розумне розпізнавання документів
- **OCR розпізнавання**: EasyOCR + Tesseract (+66% точність)
- **Попередня обробка**: контраст, шум, нахил
- **Оцінка якості**: поради для кращого фото
- **Багатосторінковість**: автоматичне об'єднання сторінок

### 🤖 Інтелектуальна класифікація
- **5 типів документів**: юридичні, сервісні, чеки, особисті, зображення
- **Автоматичне визначення**: чи це юридичний лист
- **Попередження**: якщо документ не юридичний

### 🌐 Професійний переклад
- **Юридичний словник**: 35+ термінів з виправленнями
- **Пост-обробка**: виправлення помилок перекладу
- **Кілька сервісів**: Google Translate + LibreTranslate
- **Кешування**: 7 днів для швидкості

### ⚖️ Локальна база законів
- **18 кодексів**: BGB, SGB, ZPO, AO, StGB тощо
- **67+ параграфів**: з перекладом українською
- **8 організацій**: Jobcenter, Finanzamt, Inkasso...
- **40+ ситуацій**: повні описи з наслідками

### 📝 Генерація відповідей
- **Двомовні відповіді**: українська + німецька
- **Професійний тон**: готові шаблони
- **Посилання на закони**: § 59 SGB II, BGB § 286...
- **Поради**: що робити далі

### 👤 Користувацький сервіс
- **Реєстрація**: збереження налаштувань
- **Історія листів**: перегляд попередніх аналізів
- **Налаштування**: 4 мови інтерфейсу
- **Підтримка**: контактна інформація
